# Часы накальных индикаторах ИВ-13 (IV-13 Nixie Clock)

![IV-13](data/iv13.jpg)

Часы на шести газоразрядных индикаторах ИВ-13 с управлением на ESP32.

## Возможности

- **Время** — NTP-синхронизация, поддержка часовых поясов
- **Криптовалюты** — отображение курса BTC/ETH (CoinGecko API)
- **Погода** — температура/давление/влажность (OpenWeatherMap, BME280, Narodmon.ru)
- **Датчики** — BME280 (T/P/H/altitude), VEML7700 (освещённость)
- **Автосмена режимов** — настраиваемое время показа для каждого режима
- **Анимации** — эффекты смены изображения (поэлементное затухание/появление)
- **Яркость** — автоматическая регулировка по датчику освещённости
- **Веб-интерфейс** — настройка WiFi, NTP, API-ключей, языка, анимаций (GyverPortal)
- **OTA-обновления** — прошивка по WiFi (mDNS: `iv13-clock.local`)
- **Индикация потери WiFi** — при обрыве связи на дисплей выводится "wifi"
- **Языки** — русский, английский

## Аппаратная часть

- **ESP32** (NodeMCU-32S)
- **6 × ИВ-13** (накальные индикаторы / нумитроны)
- **TPIC6B595** — сдвиговый регистр с силовыми ключами (SPI)
- **Драйвер анодов** — ШИМ (канал 0, 15 кГц, 8 бит) для регулировки яркости
- **BME280** — температура, влажность, давление (I²C)
- **VEML7700** — цифровой датчик освещённости (I²C)
- **DS1307** — RTC (резервный, опционально)

## Сборка и прошивка

### PlatformIO

```bash
# Сборка
pio run

# Прошивка по USB
pio run --target upload --upload-port /dev/ttyACM1

# Прошивка по WiFi (OTA)
pio run --target upload --upload-port iv13-clock.local
```

### Зависимости

Библиотеки подключаются автоматически через `lib_deps` в `platformio.ini`:
- GyverPortal — веб-интерфейс
- GyverBME280 — датчик BME280
- Adafruit VEML7700 — датчик освещённости
- GyverSegment — отображение символов
- FileData — хранение настроек
- Arduino_JSON — парсинг JSON
- NTPClient — синхронизация времени
- ArduinoMD5 — MD5-хеш (Narodmon API)

## Настройка

Первый запуск — устройство создаёт точку доступа `IV13-AP`. Подключитесь к ней, откройте `http://192.168.4.1` и настройте:
- Подключение к WiFi
- Часовой пояс и NTP-сервер
- API-ключи OpenWeatherMap / Narodmon.ru
- Язык интерфейса
- Режимы и анимации

## Пин-аут (SPI)

| Сигнал | ESP32 GPIO |
|--------|------------|
| HSPI_SCLK | GPIO14 |
| HSPI_MOSI | GPIO13 |
| HSPI_MISO | GPIO12 |
| HSPI_SS | GPIO15 |
| ШИМ яркость | GPIO2 |
